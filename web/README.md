# Performa Frontend

## dependencies
- node.js@latest (install the latest node.js)
- npm@latest (install the latest npm)
- vue3@latest https://vuejs.org/guide/quick-start.html
- element-plus https://element-plus.org/en-US/guide/quickstart.html

## quick start
```shell
$ npm install
$ npm run dev
```
